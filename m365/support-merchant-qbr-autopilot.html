Support for Merchant QBR AutoPilot

Last updated: June 5, 2026

If you need help with Merchant QBR AutoPilot, use the support channels below. This page is the official support resource for this app.

1. Contact Support

  • Email: support@intellithing.com
  • Support Hours: Monday-Friday, 9:00 AM-6:00 PM PT
  • Target Initial Response: within 1 business day

2. What to Include in Your Request

To speed up troubleshooting, include:

  • App name and environment (for example, Merchant QBR AutoPilot).
  • Timestamp of the issue and your time zone.
  • Error message or screenshot.
  • Request ID / Conversation ID from agent debug info (if available).
  • Steps to reproduce the issue.

3. Common Issues

  • Scorecard generation fails: verify storage configuration and deployment health.
  • Sample merchant lookup errors: check API endpoint health and data file availability.
  • Authorization failures: confirm Entra app permissions and tenant/app registration state.

4. Service Status

If there is a known incident affecting Merchant QBR AutoPilot, we will provide updates through the support email channel and your designated administrator contact.

5. Escalation

For urgent production-impact issues, email support@intellithing.com with subject prefix [URGENT][Merchant QBR AutoPilot].

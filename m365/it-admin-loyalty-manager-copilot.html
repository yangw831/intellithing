What it does

Loyalty Manager Copilot AI is a Copilot agent that helps loyalty managers, CRM teams, and operations staff diagnose loyalty program issues, analyze program performance, and prepare executive-ready reporting — all from a guided conversation inside Microsoft 365 Copilot. No separate backend account or subscription is required.

Users authenticate with their existing Microsoft 365 identity. The agent uses sample data when live integrations are not configured and is transparent about this at every step.

Who uses it

Loyalty program managers, CRM and retention leads, Shopify store owners, operations teams, and marketing leaders who work within Microsoft 365.

Permissions requested and why

PermissionTypeWhy it's needed
identity Microsoft 365 standard Identifies the signed-in user to personalize responses
messageTeamMembers Microsoft 365 standard Allows the agent to send response cards within Teams conversations

No Microsoft Graph write permissions are requested. The agent cannot read email, calendar, files, or contacts. It cannot write to any Microsoft 365 service. There is no custom backend API — all processing occurs within Microsoft 365 Copilot.

Where data lives

  • All data processing occurs within Microsoft 365 Copilot and your organization's Microsoft tenant.
  • No user data is transmitted to or stored by IntelliThing. This is a declarative agent with no external backend.
  • Any data uploaded by users (such as CSV files) is processed within the Microsoft 365 Copilot conversation context and governed by your tenant's Microsoft data retention policies.
  • Authentication is handled entirely by Microsoft Entra ID. IntelliThing never sees user credentials or tokens.

How to approve

When a user clicks Request approval on the app listing, the request appears in the M365 Admin Center.

Step 1: Sign in to admin.cloud.microsoft

Step 2: Navigate to either:

  • Agents → All agents → Requests (recommended for Copilot agents)
    https://admin.cloud.microsoft/#/agents/all/requested
  • Settings → Integrated apps → Requested apps (also works)
    https://admin.cloud.microsoft/#/Settings/IntegratedApps

Step 3: Find Loyalty Manager Copilot AI by IntelliThing, Inc. and select Accept request.

Once approved, the app becomes available to the requesting user. You can also deploy it to a broader group from the same screen.

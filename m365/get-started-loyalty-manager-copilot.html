Getting Started with Loyalty Manager Copilot AI

1. Open the Agent

Launch Loyalty Manager Copilot AI in Microsoft 365 Copilot and sign in with your work account.

2. Choose an Issue to Diagnose

Start by telling the agent what you want to work on — for example, missing points, a redemption problem, or a campaign that did not trigger. Use plain language; the agent will guide you through the right workflow.

3. Answer the Workflow Questions

The agent will ask a few questions about your data sources, approval preferences, and how you want the workflow to behave. These shape the recommended action.

4. Review the Proposed Action

The agent presents what it found and what it would do as an operator. If live data is not connected, it will use realistic sample data and clearly indicate this.

5. Approve and See Results

Confirm the proposed action when prompted. The agent simulates execution and shows a before/after summary so the workflow feels concrete.

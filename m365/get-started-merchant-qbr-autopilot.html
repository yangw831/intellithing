Getting Started with Merchant QBR AutoPilot

Find the app in Microsoft 365: Search for Merchant QBR AutoPilot in Microsoft Teams Apps, Microsoft Marketplace, or the Microsoft 365 Copilot agent catalog. If your tenant requires approval, click Request approval and send the IT admin guide to your Microsoft 365 administrator.
Merchant QBR AutoPilot

1. Find and Open the Agent

In Microsoft Teams or Microsoft 365 Copilot, search for Merchant QBR AutoPilot. Add the app if it is available, or click Request approval if your organization requires IT admin approval.

2. Review the Input Schema

Ask for sample data schema first to confirm required CSV fields and data shape.

3. Choose a Merchant

List sample merchants or provide a merchant from your uploaded CSV.

4. Generate Scorecard

Run scorecard generation and confirm the action when prompted.

5. Download and Review

Use the returned `downloadUrl` before the link expires.

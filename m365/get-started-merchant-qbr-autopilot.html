Getting Started with Merchant QBR AutoPilot

1. Open the Agent

Launch Merchant QBR AutoPilot in Microsoft 365 Copilot and sign in with your work account.

2. Review Input Schema

Ask for sample data schema first to confirm required CSV fields and data shape.

3. Choose a Merchant

List sample merchants or provide a merchant from your uploaded CSV.

4. Generate Scorecard

Run scorecard generation and confirm the action when prompted.

5. Download and Review

Use the returned `downloadUrl` before the link expires.

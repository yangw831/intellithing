Support for Loyalty Manager Copilot AI

Last updated: June 5, 2026

If you need help with Loyalty Manager Copilot AI, use the support channels below. This page is the official support resource for this app.

1. Contact Support

  • Email: support@intellithing.com
  • Support Hours: Monday–Friday, 9:00 AM–6:00 PM PT
  • Target Initial Response: within 1 business day

2. What to Include in Your Request

To speed up troubleshooting, include:

  • App name: Loyalty Manager Copilot AI
  • Timestamp of the issue and your time zone
  • Error message or screenshot
  • Conversation ID from agent debug info (if available)
  • Steps to reproduce the issue

3. Common Issues

  • Agent does not respond as expected: try rephrasing your request or starting a new conversation.
  • Sample data shown instead of live data: this is expected when live integrations are not configured — the agent will indicate when it is using sample data.
  • App not available in your organization: your IT admin may need to approve the app first. See the IT Admin Approval Guide.

4. Service Status

If there is a known incident affecting Loyalty Manager Copilot AI, we will provide updates through the support email channel and your designated administrator contact.

5. Escalation

For urgent production-impact issues, email support@intellithing.com with subject prefix [URGENT][Loyalty Manager Copilot AI].
